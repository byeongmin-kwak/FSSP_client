## 프로젝트 소개

- 집사는 자신이 직접 살았던 집에 대해 리뷰를 작성하고 다른 사람들과 공유하여 집을 구할 때 도움을 주는 서비스입니다.
- 집의 장점, 단점, 키워드들을 입력하여 리뷰를 작성할 수 있습니다.
- 지도를 통해 리뷰를 볼 수 있습니다.
- 주소 검색을 통해 해당 집의 건물정보를 알 수 있고, 리뷰를 모아볼 수 있습니다.
- 관심 있는 집은 관심건물에 등록하여 모아볼 수 있습니다.
